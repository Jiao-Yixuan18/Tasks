## Github 使用

* 如何创建自己的仓库及在仓库中创作/上传文档
  
  1、进入Github中的Repositories

  2、点击左上角绿色的“NEW”

  3、输入仓库名称、描述后点击下方的Create repository即完成

  4、在所创建的仓库这里点击Add File即可在该仓库里创作文档(Create new file)或者上传文档(Upload files)
  
  （在这里创建文档时可直接用Markdown）

  5、写完内容后点击右上角的Commit changes

 * Github中的常用功能

| 词语         | 作用                                       |
| ------------ | :----------------------------------------- |
| Fork         | 复制别人的仓库到自己的账号                 |
| Clone        | 把远程仓库下载到本地电脑，方便离线编辑代码 |
| Pull Request | 把Fork后修改的内容，申请合并回原项目       |
| Star         | 收藏项目                                   |

等等
