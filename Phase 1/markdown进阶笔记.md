# 基本工作原理

Markdown 源文件 (.md) → Markdown 解析器 → HTML 文档 → 浏览器渲染

### 1. 编写 Markdown 源文件

一般用Typora和Github，创建 .md 文件，使用 Markdown 语法编写内容

### 2. Markdown 解析器处理

解析器读取 Markdown 文件，识别语法元素并转换为对应的 HTML

### 3. 浏览器渲染


生成的 HTML 在浏览器中显示为格式化的文档
