# Github

### Github是做什么的

Github 是全球最大的**代码托管平台**，核心用于版本控制（Git 工具）、团队协作开发，还能**存储、分享开源项目、管理技术文档**。

简单说，开发者用它**存代码、回溯修改记录**，团队成员可**共同编辑项目**，普通用户能**下载免费开源工具（比如编程库、小软件）**。

作为学生党，我在这里可以**免费获取很多学习资源，打破信息差**。

### Github中的词语是做什么的

举例：

| 词语         | 作用                                       |
| ------------ | :----------------------------------------- |
| Fork         | 复制别人的仓库到自己的账号                 |
| Clone        | 把远程仓库下载到本地电脑，方便离线编辑代码 |
| Pull Request | 把Fork后修改的内容，申请合并回原项目       |
| Star         | 收藏项目                                   |
