# 问题解决记录2

备注：Q代表问题（question）,S代表解决方法（solution）

### 1
* Q:检查一下 SSH Key 是否安装成功时，出现被拒绝连接是怎么回事？怎么办？
* S：SSH连接GitHub(默认使用22端口)，有时网络阻止了22端口，但443端口是开放的。将教程中            ssh -T git@github.com   换成   ssh -T -p git@ssh.github.com
    出现一个需要输入的选择题，输入yes，回车
    出现Hi,用户名!You 've successfully authenticated...
    就成功了，然后再设置全局的用户名和邮箱
    教程：https://zhuanlan.zhihu.com/p/337959303

### 2
* Q：GitHub打不开，页面出现乱码,怎么回事？怎么办？

* S：DNS污染，寻求了mentor的帮助，通过重启电脑，清理了系统内部的临时状态。能通过重启解决DNS污染的问题，说明DNS污染可能发生在本地或局域网层面。

### 3 Git初始化提交失败
Q:用命令git add告诉Git，把文件添加到仓库时，出现Author identity unknown......
S:Git无法自动检测到我的邮箱地址，Git需要我设置用户名称和邮箱地址

安装好Git后，还需要最后一步设置，在命令行输入：

$ git config --global user.name "Your Name"

$ git config --global user.email "email@example.com"
