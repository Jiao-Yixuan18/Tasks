# 页面编后感

***
### 我的项目

我创建了一个关于家乡传统文化的网页，以像坐公交车一样的方式，一站一站地展开网页内容。

网页名叫“齐鲁文化”，整体分为六个部分：主页，儒家文化，泰山文化，民俗艺术，美食文化，关于。

其中，主页算是一个引子，而关于包含了这个网页的大概情况及我做这个网页的初衷。

说来惭愧，收集资料的过程中我也发现了很多我原本不知道的文化内容。所以在这个过程中，实际上我既是介绍者，又是传播者。

网页最终算是完成了我的基本构想，虽然在制作过程中也割舍了一些内容（比如意见收集的表单，没有后端，没法存到数据库），但我想呈现的也算基本实现了。

### 创建过程

制作之前我现在草稿纸上写了写我的网页大约包含哪些内容，收集了相应的内容的文本资料，画了画每个小网页的大体“形象”（网页布局）。这个工作完成之后我的创建过程就有目的性多了。比如创建主页时，我要先写标题和导航，那我就直接去搜B站上”如何创建导航“类似的视频，并配合菜鸟教程学习，每一个板块都差不多沿用这样的学习模式，然后一步一步完成我的网页。


HTML：我用html简单构建了网页的骨架，比如主页的头部标题，导航条，前言等内容。

CSS：在HTML基础上，我主要在HTML头部添加一些类选择器，对页面进行修饰：

1.大量的卡片使内容层次分得更加清晰分明。

2.按钮和导航在按前、按中过程加了颜色区分，让浏览者方便得知是否点到了相应的链接。

3.给卡片加阴影实现立体效果，使网页更加美观。

4.为网页添加背景，给网页插入一些相应内容的图片，使网页画面更加生动具象，同时凸显我的网页主题——齐鲁文化，比如大标题的背景是“鲁锦”。

5.标题和段落用了不同的颜色区分，但还要协调不违和。

### 难题
最大的问题：属性太多了，用到的时候想不起来，要反复回菜鸟教程再学习

解决方式：在时间有限的情况下，我改变了一开始挨个看菜鸟教程的学习方式，而是从B站上找了一些时长比较短的、基础的学习HTML CSS JS视频，还收藏了一些针对某个效果的教程，菜鸟教程逐渐从学习的”主战场“变成了我学习前端的“字典“。

### 优化和补完
这十天的学习过程感觉自己还是很不熟练，目前这个网页还是挺单调的，基本上每个网页都用的一套模式。如果以后进一步精进自己的技术，我想再完善并丰富一下，加一点动画之类的效果，添加意见收集的表单等等。
