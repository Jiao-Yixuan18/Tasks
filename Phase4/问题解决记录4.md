## 3.图片设置

##### 7.如何让logo放在背景图片上，并居中

把logo图片设置为块级元素，position设为absolute，使logo绝对定位，脱离文档流
*  脱离文档流：
  简单说，脱离文档流就是元素不再遵循HTML默认的“从上到下、从左到右”的排版规则，不会占据原本的空间，也不会影响其他元素的位置。

把文档流想象成一页空白的笔记本：

- 在文档流里：文字/图片会像写字一样，一行行排，每个元素都占自己的格子，后面的内容会乖乖排在它后面；
- 脱离文档流后：这个元素就像“飘”在了页面上，原来的格子会被后面的元素直接占掉，它自己的位置全靠  position / float  这类属性控制。

常见会让元素脱离文档流的情况

1.  position: absolute、 position: fixed ；
2.  float: left  /  float: right ；
3. 特殊的  display: none （直接隐藏，彻底不占空间）。

给  .logo  加position: absolute ，它就脱离了文档流：

- 页面里原本该留给logo的位置会被后面的内容（比如那些三角形、文字）直接填上；
- 所以需要用  left: 50% + transform  手动定位，而不是靠文档流的规则让它居中。

## 文字居中

##### 10.如何让无序列表居中

把无序列表设成行内块元素display：inline-block，通过父容器text-align：center使其居中

##### 11.为什么设置列表居中成功后，无序列表标志与文字之间有很大空隙,如何解决
> 第一步：先定位核心问题

浏览器给  <ul>  预设了  padding-left （通常是 40px 左右），且列表标记默认在  <li>  外部（ list-style-position: outside ），这就会让黑点和文字隔得很远。

> 第二步：两种修复方案

* 方案1：调整标记位置

把列表标记放到  <li>  内部，再清除多余内边距，是最贴合你需求的方式：

```css
ul {
  padding-left: 0; /* 清除ul默认左内边距 */
  margin: 0 auto;  /* 保持列表整体居中（延续之前的居中需求） */
  text-align: center; /* 让li里的文字居中 */
}
li {
  list-style-position: inside; /* 把黑点放到li内容内部 */
  list-style-type: disc; /* 保留小黑点样式 */
  margin: 8px 0; /* 可选：调整列表项之间的行距 */
}
```


* 方案2：微调内边距（保留标记在外部）

如果想让黑点仍在  <li>  外侧，只缩小间距：

```css
ul {
  padding-left: 20px; /* 把默认40px改成更小的值，按需调 */
  margin: 0 auto;
  text-align: center;
}
li {
  list-style-position: outside; /* 默认值，可省略 */
  text-align: left; /* 让li内文字左对齐（避免黑点和文字都居中错位） */
  margin-left: auto;
  margin-right: auto;
  display: table; /* 让li自适应宽度并居中 */
}
```

html

<!-- HTML：问答列表结构 -->
<div class="faq-container">
  <!-- 第一个问答项 -->
  <div class="faq-item">
    <div class="faq-question" id="q1">
      Q：加入需要具备哪些基础？

  <!-- 第二个问答项 -->
  <div class="faq-item">

      Q：如何报名加入？
      <span class="toggle-icon">+</span>
    </div>
    <div class="faq-answer">
      A：非新思路统招只有通过发送邮件自我介绍后，收到邮件回执后才会被允许进群，在读学生都可邮件到newthread_geek@outlook.com报名加入（信中需备注个人基本信息以及意向方向，此招新活动为长期招新）。
    </div>
  </div>
</div>

<style>
/* CSS：样式+收起状态默认隐藏 */
.faq-container {
  width: 80%;
  margin: 20px auto; /* 整体居中 */
}

.faq-item {
  margin: 10px 0;
  border: 1px solid #ddd;
  border-radius: 8px;
  overflow: hidden;
}

.faq-question {
  padding: 12px 16px;
  background-color: #f5f5f5;
  cursor: pointer;
  font-weight: bold;
  display: flex;
  justify-content: space-between; /* 文字和+/-图标分开 */
  align-items: center;
}

.faq-answer {
  padding: 0 16px;
  height: 0; /* 默认收起，高度为0 */
  overflow: hidden;
  transition: height 0.3s ease; /* 展开/收起动画 */
  background-color: #fff;
}

/* 展开时的样式（JS动态添加） */
.faq-answer.show {
  padding: 12px 16px;
  height: auto; /* 自适应内容高度 */

.toggle-icon {
  font-size: 18px;
  color: #2da44e; /* 匹配你按钮的绿色 */

<script>
// JS：点击问题切换展开/收起状态
// 1. 获取所有问题元素
const questions = document.querySelectorAll('.faq-question');

// 2. 给每个问题绑定点击事件
questions.forEach(question => {
  question.addEventListener('click', function() {
    // 找到当前问题对应的回答
    const answer = this.nextElementSibling;
    // 找到切换图标
    const icon = this.querySelector('.toggle-icon');

    // 切换展开/收起状态
    answer.classList.toggle('show');
    
    // 切换图标（+/-）
    icon.textContent = answer.classList.contains('show') ? '-' : '+';
  });
});
</script>
